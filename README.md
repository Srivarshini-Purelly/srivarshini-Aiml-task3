# Task 3 - Linear Regression

## Objective
Implement Simple and Multiple Linear Regression using Scikit-Learn.

## Steps Performed
1. Loaded and preprocessed the dataset.
2. Split data into training and testing sets.
3. Trained a Linear Regression model.
4. Evaluated the model using:

   - MSE
   - R² Score
5. Visualized Actual vs Predicted values.
6. Interpreted feature coefficients.

- Pandas
